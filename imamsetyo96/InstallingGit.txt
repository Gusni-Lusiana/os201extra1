==============================================================================

I. DOWNLOAD GIT

Silahkan download Git, dan sesuaikan dengan sistem operasi masing-masing
https://git-scm.com/downloads

==============================================================================
II. INSTALL GIT

1.  Disini saya menggunakan package manager Homebrew dalam proses instalasi Git

2.  Homebrew
    https://brew.sh/

3.  Lakukan proses instalasi Homebrew menggunakan terminal
    == /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

4.  Setelah proses instalasi selesai, install Git melalui Homebrew pada terminal
    == brew install git

4.  Cek versi dari Git yang Digunakan
    == git --version
    == Hasil:
    git version 2.15.0


==============================================================================
III. REGISTRASI GIT

1.  Set global konfigurasi pada Git dan sesuaikan dengan nama serta email masing-masing
    == git config --global user.name "Imam Setyo Santoso"
    == git config --global user.email "imamsetyosantoso@gmail.com"

2.  Lakukan pengecekan pada konfigarsi Git dengan perintah
    == git config --list
    == Hasil:
    core.excludesfile=~/.gitignore
    core.legacyheaders=false
    core.quotepath=false
    mergetool.keepbackup=true
    push.default=simple
    color.ui=auto
    color.interactive=auto
    repack.usedeltabaseoffset=true
    alias.s=status
    alias.a=!git add . && git status
    alias.au=!git add -u . && git status
    alias.aa=!git add . && git add -u . && git status
    alias.c=commit
    alias.cm=commit -m
    alias.ca=commit --amend
    alias.ac=!git add . && git commit
    alias.acm=!git add . && git commit -m
    alias.l=log --graph --all --pretty=format:'%C(yellow)%h%C(cyan)%d%Creset %s %C(white)- %an, %ar%Creset'
    alias.ll=log --stat --abbrev-commit
    alias.lg=log --color --graph --pretty=format:'%C(bold white)%h%Creset -%C(bold green)%d%Creset %s %C(bold green)(%cr)%Creset %C(bold blue)<%an>%Creset' --abbrev-commit --date=relative
    alias.llg=log --color --graph --pretty=format:'%C(bold white)%H %d%Creset%n%s%n%+b%C(bold blue)%an <%ae>%Creset %C(bold green)%cr (%ci)' --abbrev-commit
    alias.d=diff
    alias.master=checkout master
    alias.spull=svn rebase
    alias.spush=svn dcommit
    alias.alias=!git config --list | grep 'alias\.' | sed 's/alias\.\([^=]*\)=\(.*\)/\1\	 => \2/' | sort
    include.path=~/.gitcinclude
    include.path=.githubconfig
    include.path=.gitcredential
    diff.exif.textconv=exif
    credential.helper=osxkeychain
    user.name=Imam Setyo Santoso
    user.email=imamsetyosantoso@gmail.com
    core.repositoryformatversion=0
    core.filemode=true
    core.bare=false
    core.logallrefupdates=true
    core.ignorecase=true
    core.precomposeunicode=true

3.  Pastikan bahwa global username dan global email sudah sesuaikan
    == user.name=Imam Setyo Santoso
    == user.email=imamsetyosantoso@gmail.com
